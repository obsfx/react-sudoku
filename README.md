# React Sudoku

A sudoku game built using React and TypeScript.

https://react-sudoku-sigma.vercel.app/

![](https://raw.githubusercontent.com/obsfx/react-sudoku/main/preview.png)



### To run

+ `git clone git@github.com:obsfx/react-sudoku.git  `
+ `cd react-sudoku`
+ `npm install`
+ `npm run start`
